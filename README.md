# Project Phase 2

##  Project Description
Course project developed using React.js, including reusable components, multiple pages, product management, ratings, shopping cart functionality, and responsive user interface design.

##  Setup Instructions
To run this project locally, follow these steps:

1. Clone the repository:
bash
git clone https://github.com


2. Go to the project folder:
bash
cd renad_project


3. Install dependencies:
bash
npm install


4. Start the app:
bash
npm start
